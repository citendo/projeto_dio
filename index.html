Atributos das Tags

Toda tag HTML pode ter Atributos, e o que são atributos? Eles server para prover infomações adicionais para as tags.
Algumas Tags precisam de atributo, outras não, a Tag P < p >, não precisa de atributo, já a tag IMG que é o exemplo a baixo precisa...
A tag IMG tem o atributo SRC que é usado para buscar a imagem dentro do projeto.
A forma de usar é < img src="#" > .



Outro atributo da tag IMG é o atributo ALT.
< img src="####" alt="#####" >



O atributo ALT é utilizado para caso a imagem vier a "quebrar" e sendo necessário colocar uma descrição para a imagem, o navegadores também usam a tag ALT para descrever imagens para usuários com deficiencia visual, onde a ferramenta de descrição usa o ALT para descrever a imagem para o usuário com deficiencia.
Existem várias outros Atributos dentro de várias outras Tags, esse são só alguns exemplos...
Em resumo algumas tags precisam de atributos, como no caso da tag IMG e outras não, como na tag P, que os atributos são opcionais. Voltar ao menu.

Listas Ordenadas e Não Ordenadas

Aqui é bem simples, para fazer uma lista ordenada usa OL(OL significa Orderned List... que passando para o português fica Lista ordenada... Tudo faz sentido) como tag e LI para definir as linhas.
< OL >
< LI > item < /LI>
< /OL >
E o exemplo fica assim:

  1. Frieren e a Jornada para o além
  2. Fullmetal Alchemist Brotherhood
  3. Bocchi the Rock

Esses são nomes de animações japonesas.. espero que não se importem
E agora para listas não ordenadas usa a tag UL (UL significa Unordered List... que em português significa aquilo que você já imagina Lista não Ordenada)
< UL >
< LI > item < /LI>
< /UL >

  • Fullmetal Alchemist Brotherhood
  • Bocchi the Rock
  • Frieren e a Jornada para o além
